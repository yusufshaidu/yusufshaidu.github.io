## 🛠️ Software & Tools

I develop and utilize a comprehensive suite of computational tools for materials science, machine learning, and atomistic simulations. Below is an overview of the software I work with regularly.

---

## 🖥️ Atomistic Simulation Packages

| Tool | Purpose | Use Case |
|------|---------|----------|
| **VASP** | DFT electronic structure calculations | Band structure, DOS, crystal optimization |
| **Quantum ESPRESSO** | Open-source DFT framework | High-throughput calculations, pseudopotential development |
| **ORCA** | Quantum chemistry package | Molecular properties, excited states |
| **LAMMPS** | Molecular dynamics engine | Large-scale MD simulations, force field testing |
| **ASE** | Atomic Simulation Environment | Python interface for workflows, structure manipulation |
| **Phonopy** | Phonon calculations | Vibrational properties, thermal effects |

## 🧠 Machine Learning Interatomic Potentials (MLIPs)

### Frameworks & Models
- **MACE** – Equivariant message passing for robust potentials
- **e3nn** – Equivariant neural networks with geometric deep learning
- **PANNA** – Scalable neural network potentials (collaborator framework)
- **BACENET** – Efficient deployment of neural network potentials

### Key Features
- Physics-informed architectures
- Equivariance to atomic rotations and permutations
- Scalable to large systems
- Efficient inference for MD simulations

## 🔧 Specialized Workflows & Tools

### Materials Discovery & Analysis
- **twist-layer toolkit** – Automated analysis of twisted bilayer systems and moiré heterostructures
- **SA-RSS workflow** – Simulated annealing with random structure search for crystal discovery
- **CSP workflows** – Crystal structure prediction pipelines

### Data Generation & Processing
- Automated DFT workflow orchestration
- High-throughput data curation for MLIP training
- Structure validation and filtering
